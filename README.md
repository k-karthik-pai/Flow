# Flow

Flow is a minimalist Chrome extension that blocks distracting websites based on your daily focus goal.

## Installation
1. Clone this repository.
2. Go to `chrome://extensions/`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the `Flow/` folder.

## Setup
- **AI Blocking**: Add your Gemini API key in the extension settings to enable AI-powered blocking and appeals.
- **Manual Blocking**: Add sites directly to the blocklist in settings.

## Privacy
All data, including API keys and goals, is stored locally on your device.
