# ⚡ Flow

**Flow** is a professional-grade productivity extension designed to protect your focus. It replaces mindless browsing with intentional work by blocking distracting websites based on your specific goal for the day.

---

## 🏛️ Core Principles

- **Intentionality First** — Every day starts with a single question: *"What are you focusing on today?"*
- **No Cheating** — No pause buttons. No easy bypasses. Commit to your goal or submit a valid AI-adjudicated appeal.
- **Privacy by Design** — All data, including your API key and browsing stats, stays locally on your device.
- **Non-Intrusive** — No "new tab" hijacking. Flow runs silently in the background and only intervenes when you drift off-course.

## ✨ Features

- **AI-Driven Blocking** — Gemini analyzes your daily goal and dynamically identifies which sites align with your intent.
- **AI Appeal System** — Think a site was blocked by mistake? Explain your reasoning to the AI to request a temporary unblock.
- **Professional UI** — A clean, Google-inspired interface with native System, Light, and Dark mode support.
- **Manual Control** — Standard manual blocklists and whitelists for users who prefer total control.
- **Zero-Distraction Startup** — Prompts you for your goal only once upon browser startup (for AI-enabled users).

## 🚀 Setup

### 1. Installation
1. Clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer Mode** (top right).
4. Click **Load unpacked** and select the `Flow/` project folder.

### 2. Enable AI Features (Optional)
To unlock AI-powered blocking and appeals, provide your own Gemini API key:
1. Get a free key at [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Open **Flow Settings** (via the extension popup or options page).
3. Paste your key and save.

## 🛠️ Technology

- **Manifest V3** — Built using the latest Chrome extension standards.
- **Declarative Net Request** — High-performance, privacy-preserving network filtering.
- **Gemini 1.5 Flash** — Lightweight, low-latency AI for real-time goal analysis.
- **Vanilla JS & CSS** — Zero dependencies for maximum speed and security.
